Post Authentication

domain/introduceyourself/login

For authentication enter your username and password with base64_encode and you will get a token for the header.

Example Code (PHP):

base64_encode($username. ":" . $password);

Important Note

the token is valid for 5m only, after 5m you need to generate a new token with same username and password

Example Request Authentication

curl --location --request POST 'domain/introduceyourself/login' \
